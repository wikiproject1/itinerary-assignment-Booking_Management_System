## 1. Introduction

The Travel Management System is a web-based solution designed for managing travel itineraries and guide assignments. This system helps streamline the process of managing group travels and guide assignments.

### Key Benefits
- Efficient itinerary tracking
- Real-time notifications for upcoming arrivals
- Guide assignment management
- Financial tracking
- User-friendly interface

## 2. System Overview

### Login Page
![Login Page]
The secure login page provides access to authorized administrators only. Features include:
- Secure authentication
- Password protection
- Professional design with navy blue gradient
- Responsive layout for all devices

### Dashboard
![Dashboard]
The main dashboard provides:
- Upcoming arrivals notifications (next 3 days)
- Total active bookings
- Financial summaries
- Quick access to all features

## 3. User Roles

### Administrator
- Full system access
- Manage itineraries
- Assign guides
- View reports
- Monitor upcoming arrivals

## 4. System Features

### 4.1 Itinerary Management
![Itinerary Management]
- View all itineraries
- Track group details
- Monitor payment status
- Update trip status
- Record starting and final destinations
- Manage arrival and departure times

### 4.2 Guide Assignment
![Guide Assignment]
- View all guides
- Manage guide information
- Track guide assignments
- Update guide status
- Record vehicle information

### 4.3 Reports
![Reports]
- View financial summaries
- Track payments and deposits
- Monitor booking trends
- View completion status

### 4.4 Notification System
![Notifications]
- Real-time upcoming arrival alerts
- Shows next 3 upcoming groups
- Displays guide assignment status
- Quick access to itinerary details
